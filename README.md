# WebCraft

A browser-based voxel game inspired by Minecraft, built with TypeScript, 
Babylon.js, and WebGL. Runs entirely in the browser with no downloads required.

## About
WebCraft is an independent fan project inspired by Minecraft. It is not 
affiliated with, endorsed by, or associated with Mojang Studios or Microsoft.

Minecraft assets used in this project are property of Mojang Studios and 
are used under their fan project usage guidelines.
